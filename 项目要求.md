# 企业级员工管理系统 - 项目要求

## 一、项目概述

本项目是一个企业级员工管理系统，基于 Spring Boot 框架构建，提供用户认证和员工信息管理功能。

## 二、技术栈

| 分类 | 技术 | 版本 |
|------|------|------|
| 语言 | Java | 21 |
| 框架 | Spring Boot | 3.2.x |
| 数据库 | MySQL | 8.0+ |
| ORM | Spring Data JPA | 3.2.x |
| 安全 | Spring Security | 6.2.x |
| 模板引擎 | Thymeleaf | 3.1.x |
| UI框架 | Bootstrap | 5.3.x |

## 三、功能需求

### 3.1 用户认证模块

| 功能 | 描述 |
|------|------|
| 用户注册 | 提供注册页面，支持用户名、密码注册 |
| 用户登录 | 提供登录页面，支持用户名和密码验证 |
| 密码加密 | 使用 BCrypt 算法加密存储密码 |

### 3.2 员工管理模块

| 功能 | 描述 |
|------|------|
| 展示全部员工 | 列表展示所有员工信息 |
| 按名称查询 | 支持根据员工姓名模糊搜索 |
| 添加员工 | 提供表单添加新员工 |
| 删除员工 | 支持删除指定员工 |
| 修改员工 | 支持编辑员工信息 |

### 3.3 部门管理模块

| 功能 | 描述 |
|------|------|
| 部门列表 | 展示所有部门信息 |
| 部门关联 | 员工关联所属部门 |

## 四、数据模型

### 4.1 用户表 (users)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键，自增 |
| username | VARCHAR(50) | 登录名称，唯一 |
| password | VARCHAR(255) | 加密后的密码 |
| email | VARCHAR(100) | 邮箱 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 4.2 员工表 (employees)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键，自增 |
| name | VARCHAR(50) | 姓名 |
| gender | VARCHAR(10) | 性别 |
| age | INT | 年龄 |
| phone | VARCHAR(20) | 电话 |
| position | VARCHAR(50) | 职位 |
| hire_date | DATE | 入职时间 |
| salary | DECIMAL(10,2) | 薪水 |
| department_id | BIGINT | 部门ID（外键） |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 4.3 部门表 (departments)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键，自增 |
| name | VARCHAR(50) | 部门名称 |
| description | VARCHAR(255) | 部门描述 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

## 五、页面设计

### 5.1 登录页面 (login.html)

- 登录名称输入框
- 密码输入框
- 登录按钮
- 注册链接

### 5.2 注册页面 (register.html)

- 用户名输入框
- 密码输入框
- 确认密码输入框
- 邮箱输入框
- 注册按钮
- 返回登录链接

### 5.3 员工管理页面 (employee.html)

- 员工搜索框（按名称）
- 员工列表表格
- 添加员工按钮
- 编辑/删除操作按钮

### 5.4 添加/编辑员工弹窗

- 姓名、性别、年龄、电话输入框
- 职位、入职时间、薪水输入框
- 部门下拉选择
- 保存按钮

## 六、项目结构

```
employee-sys/
├── src/
│   └── main/
│       ├── java/
│       │   └── com/employeesys/
│       │       ├── controller/      # 控制层
│       │       ├── service/         # 业务层
│       │       ├── repository/      # 数据访问层
│       │       ├── entity/          # 实体类
│       │       ├── dto/             # 数据传输对象
│       │       ├── config/          # 配置类
│       │       ├── security/        # 安全认证
│       │       ├── exception/       # 异常处理
│       │       └── EmployeeSysApplication.java
│       └── resources/
│           ├── templates/           # HTML模板
│           ├── static/              # 静态资源
│           └── application.yml      # 配置文件
└── pom.xml
```

## 七、安全要求

- 使用 Spring Security 进行认证和授权
- 使用 BCrypt 加密用户密码
- 登录用户才能访问员工管理页面
- 防止 SQL 注入和 XSS 攻击

## 八、运行环境

- JDK 21+
- Maven 3.8+
- MySQL 8.0+

## 九、数据库初始化

1. 创建数据库：`CREATE DATABASE employee_sys;`
2. 配置数据库连接信息
3. JPA 自动创建表结构
